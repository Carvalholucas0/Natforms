# Nathalia Otero - Style Quiz Web App

A colorful web application for the Nathalia Otero brand to help customers discover their fashion personality type.

## Features
- Interactive questionnaire with vibrant design
- Customer data collection
- Personality type calculation based on responses
- Admin dashboard to view responses
- Mobile-responsive design
- QR code friendly
